## Multimodal_Chatbot_System


A modular and extensible project for building a **multimodal chatbot** capable of processing and understanding **text, images, and other data modalities**.  
This repository follows a clean folder structure to ensure easy development, reproducibility, and collaboration.

---

**## Project **
This project aims to develop a chatbot that integrates multiple machine learning models to interpret and respond using different types of inputs. It includes:
- Preprocessing pipelines  
- Model training and evaluation  
- Jupyter notebooks for experimentation  
- Modular Python scripts for production workflows  

---

**## Installation**
Running it is pretty easy, just: 

1] Clone the repository:

```bash
git clone https://github.com/patilgayatri22/Multimodal_Chatbot_System.git
cd Multimodal_Chatbot_System

2] Install Dependencies:
pip install -r requirements.txt

3] Explore notebooks:
Open the notebooks/ folder and run any .ipynb file.
